# CWE Cybersecurity-Instruct Data Generator



Data builder for Cybersecurity instructions based on the MITRE CWE database



## CWE data source



The raw examples are generated by downloading the CWE data dump from https://cwe.mitre.org/data/downloads.html

The specific file is "Research Concepts"

The downloaded file must be then converted to jsonl before processing



## CWE instruction types



There types of instructions are generated using the CWE database:



1. Definition - used for pre-training,

2. Q&A - various types of questions composed out of CWE data

3. Impact evaluation - test Q&A dataset based on the impact data field
